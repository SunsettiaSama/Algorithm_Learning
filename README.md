# 算法学习




## 项目结构

### 📚 学习内容
- **基础算法** - 排序、搜索、枚举等基础算法实现
- **高级算法** - 动态规划、贪心、枚举、递归、图论等算法实现
- **技巧** - 双指针、字符串处理、数据结构优化等
- **图表** - 算法学习的可视化和说明
- **More**......
### 🛠️ 工具
- **学习进度追踪系统** (`study_tracker/`) - 自建学习进度追踪器，可以扫描目前的学习进度，正在完善中

### 📖 学习资料
- **学习心得** - 算法题解题方法论和知识点总结

## 快速开始

### 查看学习进度
```bash
cd study_tracker
python main.py
```

## 主要功能

- 📊 自动追踪所有学习文件的修改时间
- 🎨 通过颜色标记直观显示复习状态（绿色/黄色/红色）
- 🔔 智能提醒需要复习的内容
- 📝 记录每日学习日志
- 📈 生成学习进度报告

## 文件说明

- `学习心得.md` - 算法解题方法论和知识区间判断
- `study_tracker/` - 学习进度追踪工具
- `基础算法/` - 包含如枚举、回溯等算法的实现
- `图论/` - 图论算法总结，如DFS、BFS解题技巧等
- `数组/` - 以数组结构为基础的算法集，包含如排序、指针等基础内容
