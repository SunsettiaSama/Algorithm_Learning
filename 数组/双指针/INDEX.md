## 双指针 (Two Pointers)
### 核心思想
- 通过两个指针在数据结构（数组/链表）上移动，减少嵌套循环，将O(n²)复杂度优化为O(n)
- 指针移动规则是核心：同向/相向/快慢，根据场景定制
### 类型分类
#### 1. 快慢指针 (Slow & Fast Pointers)
- 别名：龟兔赛跑指针
- 核心特征：两个指针同方向移动，速度不同（步长不同）
- 适用场景
  - 链表相关：判断环、找中点、找倒数第k个节点
  - 数组相关：移除重复元素、移除指定元素
- 典型例题
  - LeetCode 141: 环形链表
  - LeetCode 26: 删除有序数组中的重复项
  - LeetCode 876: 链表的中间结点
#### 2. 左右指针 (Left & Right Pointers)
- 别名：相向双指针、对撞指针
- 核心特征：两个指针从两端向中间移动（或反向）
- 适用场景
  - 有序数组：两数之和、三数之和、四数之和
  - 字符串：回文判断、反转字符串
  - 数组：二分查找（特殊的左右指针）、盛最多水的容器
- 典型例题
  - LeetCode 11: 盛最多水的容器
  - LeetCode 15: 三数之和
  - LeetCode 9: 回文数
#### 3. 滑动窗口 (Sliding Window)
- 别名：同向双指针（进阶）
- 核心特征：左右指针同向移动，形成动态窗口，维护窗口内的条件
- 适用场景
  - 子数组/子字符串：最长无重复子串、最小覆盖子串
  - 数组：长度最小的子数组、水果成篮
  - 字符串：找到字符串中所有字母异位词
- 典型例题
  - LeetCode 3: 无重复字符的最长子串
  - LeetCode 76: 最小覆盖子串
  - LeetCode 209: 长度最小的子数组
#### 4. 分离双指针 (Separate Pointers)
- 别名：多数组双指针
- 核心特征：多个指针分别在不同数组上移动，协同处理
- 适用场景
  - 多数组：合并两个有序数组、两个数组的交集
  - 有序链表：合并两个有序链表
- 典型例题
  - LeetCode 88: 合并两个有序数组
  - LeetCode 349: 两个数组的交集
  - LeetCode 21: 合并两个有序链表
### 通用解题步骤
1. 确定指针类型：根据问题特征选择快慢/左右/滑动窗口/分离指针
2. 定义指针初始位置：如快慢指针初始都在头节点，左右指针在数组两端
3. 制定移动规则：明确指针何时移动、向哪个方向移动
4. 边界条件处理：避免越界（如链表null、数组下标越界）
5. 验证终止条件：确定循环结束的条件（如指针相遇、遍历完成）
### 核心优化点
- 时间复杂度：从暴力法O(n²)→O(n)（大部分场景）
- 空间复杂度：无需额外空间（原地操作），O(1)（除滑动窗口部分场景）
### 易错点
- 指针越界：未判断指针是否为空/是否超出数组范围
- 移动规则错误：如快慢指针步长设置不当、滑动窗口收缩条件错误
- 重复计算：如三数之和未去重、滑动窗口未正确维护窗口内状态
