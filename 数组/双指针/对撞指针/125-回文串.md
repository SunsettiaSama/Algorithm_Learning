# LeetCode 125. 验证回文串 解题总结（含全版本错误复盘）
## 一、题目核心信息
### 题目描述
给定一个字符串 `s`，验证它是否是回文串，**只考虑字母和数字字符**，忽略字母的大小写。如果是回文串返回 `True`，否则返回 `False`。
- 回文串定义：正读和反读完全相同的字符串（仅针对有效字符）；
- 关键示例：
  - 输入：`"A man, a plan, a canal: Panama"` → 输出：`True`（有效字符为`amanaplanacanalpanama`，是回文）；
  - 输入：`"race a car"` → 输出：`False`；
  - 输入：`"0P"` → 输出：`False`（`0`和`p`大小写统一后仍不相等）。

### 核心要求
1. 过滤规则：仅保留**字母（a-z/A-Z）** 和**数字（0-9）**，空格、标点、符号等全部过滤；
2. 大小写规则：忽略大小写（如`A`和`a`视为相同字符）；
3. 性能要求：原地判断（无需额外开辟大量空间），时间复杂度最优为 `O(n)`。

```python
class Solution:
    def isPalindrome(self, s: str) -> bool:
        left_index = 0
        right_index = 1  # ERROR 1
        s = s.lower() # 注：转小写正确，注释误标为ERROR
```

### 2.2 V1版本（部分修正版）

```python
class Solution:
    def isPalindrome(self, s: str) -> bool:
        num_list = [' '].extend([f'{i}' for i in range(10)])  # ERROR 1
        right_index = len(s) - 1 # ERROR：注释误标，实际正确
        s = s.lower() # ERROR：注释误标，实际正确
        while left_index < right_index:
            left_s = s[left_index]
            right_s = s[right_index]
            if left_s in num_list:  # ERROR 4
                left_index += 1
                left_s = s[left_index]
            elif right_s in num_list:
                right_index -= 1
                right_s = s[right_index]
            if not left_s == right_s:  # ERROR 5
                return False
            left_index += 1
            right_index -= 1
        return True
```

```python
class Solution:
    def isPalindrome(self, s: str) -> bool:
        left_index = 0
        right_index = len(s) - 1
        while left_index < right_index:
            left_index = skip_nums(s, index = left_index, left = left_index, right = right_index, step = 1)

                return False
            left_index += 1
            right_index -= 1

def skip_nums(s: str, index, left, right, step = 1):
    while left < right and not s[index].isalnum(): # ERROR 1/2
        index += step
    return index
```

#### 核心问题
V2版本尝试封装跳过逻辑，但函数内的循环条件和边界判断错误，且遗漏“转小写”的核心步骤，过滤逻辑仍不成立。

## 三、核心解题思路（相向双指针最优解）
### 3.1 核心原理
使用**相向双指针**（左指针从头部、右指针从尾部向中间移动），先跳过非有效字符，再比较有效字符，直到指针相遇。
- 时间复杂度：`O(n)`（每个字符最多被访问2次，左/右指针各1次）；
- 空间复杂度：`O(1)`（仅使用指针变量，原地判断）。

### 3.2 分步实现（清晰版）
#### 步骤1：初始化指针（记忆点）
```python
left = 0  # 左指针：字符串头部
right = len(s) - 1  # 右指针：字符串尾部（必记！）
```

#### 步骤2：统一转小写（记忆点）
```python
s_lower = s.lower()  # 忽略大小写，必做步骤！
```

#### 步骤3：封装跳过非有效字符的辅助函数
将重复的跳过逻辑封装，避免嵌套循环杂乱：
```python
def _skip_non_alnum(s: str, ptr: int, boundary: int, step: int) -> int:
    """
    跳过非字母/数字字符，返回有效字符的指针位置
    :param s: 小写后的字符串
    :param ptr: 当前指针位置
    :param boundary: 另一指针位置（边界，避免越界）
    :param step: 移动步长（左指针+1，右指针-1）
    :return: 更新后的指针位置
    """
    # 左指针：ptr < boundary 时向右跳；右指针：ptr > boundary 时向左跳
    if step == 1:
        while ptr < boundary and not s[ptr].isalnum():
            ptr += step
    else:
        while ptr > boundary and not s[ptr].isalnum():
            ptr += step
    return ptr
```

#### 步骤4：双指针循环比较
```python
while left < right:
    # 跳过非有效字符
    left = self._skip_non_alnum(s_lower, left, right, step=1)
    right = self._skip_non_alnum(s_lower, right, left, step=-1)
    
    # 指针相遇，说明所有有效字符已比较完
    if left >= right:
        break
    
    # 有效字符不相等，直接返回False
    if s_lower[left] != s_lower[right]:
        return False
    
    # 指针向中间移动
    left += 1
    right -= 1
return True
```

### 3.3 最终可运行的优化版代码
```python
class Solution:
    def isPalindrome(self, s: str) -> bool:
        # 初始化相向双指针
        left = 0
        right = len(s) - 1
        # 统一转小写，忽略大小写
        s_lower = s.lower()

        while left < right:
            # 跳过非有效字符
            left = self._skip_non_alnum(s_lower, left, right, step=1)
            right = self._skip_non_alnum(s_lower, right, left, step=-1)

            # 指针相遇则结束比较
            if left >= right:
                break

            # 核心比较：有效字符不相等则不是回文
            if s_lower[left] != s_lower[right]:
                return False

            # 指针向中间移动
            left += 1
            right -= 1

        return True

    # 封装：跳过非字母/数字字符
    def _skip_non_alnum(self, s: str, ptr: int, boundary: int, step: int) -> int:
        if step == 1:
            while ptr < boundary and not s[ptr].isalnum():
                ptr += step
        else:
            while ptr > boundary and not s[ptr].isalnum():
                ptr += step
        return ptr

# 测试验证
if __name__ == "__main__":
    sol = Solution()
    print(sol.isPalindrome("A man, a plan, a canal: Panama"))  # True
    print(sol.isPalindrome("race a car"))  # False
    print(sol.isPalindrome("0P"))  # False
```

## 四、易错点总结 & 记忆口诀
### 4.1 高频易错点（按出错概率排序）
| 易错类型       | 典型错误                     | 规避方法                                                                 |
|----------------|------------------------------|--------------------------------------------------------------------------|
| 指针初始化     | 右指针写成`1`/`len(s)`        | 死记：右指针 = `len(s) - 1`                                              |
| 指针移动方向   | 右指针向右移动（`+=1`）| 死记：相向双指针 → 左+1，右-1                                            |
| 过滤逻辑       | 手动维护过滤列表、单次跳过   | 放弃手动列表，用`isalnum()`+`while`循环跳过（覆盖所有非有效字符）|
| 大小写处理     | 遗漏`lower()`/`upper()`       | 第一步先转小写，统一比较标准                                             |
| 边界越界       | 跳过字符后直接访问字符串     | 所有字符串访问前，先判断`left < right`                                   |
| API使用        | 忘记`isalnum()`函数          | 记忆：`isalnum()` = is alphabet + number（判断字母/数字）|

### 4.2 记忆口诀（新手友好）
```
指针左右头和尾，转小写后再比对；
跳过无效用while，isalnum来判断；
字符相等指针移，不等直接返回False；
指针相遇就结束，全程牢记不越界。
```

## 五、最佳实践总结
1. **核心逻辑**：相向双指针是验证回文串的最优解，无需额外空间，时间复杂度`O(n)`，优于“过滤后反转比较”的方法；
2. **过滤逻辑**：优先使用内置`isalnum()`，避免手动维护过滤列表（易遗漏标点/符号，易出错）；
3. **代码结构**：将跳过非有效字符的逻辑封装为辅助函数，主函数只保留核心比较逻辑，提升可读性和可维护性；
4. **边界处理**：所有字符串索引访问前，必须判断指针是否在有效范围内（`left < right`），避免越界报错；
5. **核心步骤**：转小写 → 初始化指针 → 跳过无效字符 → 比较有效字符 → 移动指针（按此顺序，逻辑不混乱）。
